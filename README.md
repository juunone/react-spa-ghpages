> React SPA with CRA and deploy with gh-pages

### 🏠 [Homepage](https://juunone.github.io/react-spa-ghpages)

## Install

```sh
yarn install
```

## Development

```sh
yarn dev
```

## Build

```sh
yarn build
```

## Deploy

```sh
yarn predeploy

yarn deploy
```
